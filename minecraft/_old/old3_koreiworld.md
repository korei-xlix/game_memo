# ワールド情報

## プレイスタイル
* バージョン：1.20.0
* 難易度：ノーマル
* モード：サバイバル
* チートあり（管理用）
* 素材は自力で集める

```
-7867976668848406519
```

## 初期スポーン  
execute in minecraft:overworld run tp @p -347 100 -177  
  

** 設定コマンド **  
```
worldborder center ~ ~
worldborder set 18000 0
setworldspawn
```
