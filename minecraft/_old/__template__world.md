# このドキュメントについて <a name="aHowto"></a>
このドキュメントはMinecrafttのこれーWorldの座標メモです。  



# ワールド情報

## プレイスタイル
* バージョン：1.20.0
* 難易度：ノーマル
* モード：サバイバル
* チートあり（管理用）
* 素材は自力で集める
  

## SEED値
    
  

## 初期スポーン  
	-2 64 -35  
  

** 設定コマンド **  
```
setworldspawn
worldborder center ~ ~
worldborder set 20000 0
```
