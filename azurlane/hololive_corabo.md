# Hololive アズレンコラボ（妄想）


# このドキュメントについて <a name="aHowto"></a>
このドキュメントはアズールレーンへ Hololiveメンバーが実装された場合、
どういう感じになるのか、デュフフ妄想するものです。
個人の勝手な妄想で作成したものなので、アズールレーンやHololiveに興味ない方の閲覧はご遠慮ください。  
  
なお、ご指摘、ご提案などあれば以下マシュマロにお寄せください。
[マシュマロ](https://marshmallow-qa.com/korei_xlix)  
  

**注意事項**  
* 卒業したメンバー、引退したメンバーについても対応しようと思います。  
  ※画像は著作権がかかってるのでナシのメンバーもいます  
    おそらく著作はカバー株式会社持ちだと思うのですが、公開してないので  
  ※ごめんだけど会社の規約違反で引退した子も好きだったので入れさせて...  

* わたしがホロを見始めたのが3期生がデビューして半年後くらいなので
  それ以前で在籍してないメンバーについては対応できないと思います。  
  情報あれば追加するので、マシュマロに送ってください。  

* 「ホロスターズ」については対応するか検討しますが、
  この記事書いてる時点ではあまり見にいってないので対応は難しいと思います。  
  情報あれば追加するので、マシュマロに送ってください。
